# GetWork Flutter App

A simple Flutter application for the GetWork project.

## Getting Started

### Prerequisites
- Flutter SDK (3.0.0 or higher)
- Dart SDK
- Android Studio / VS Code with Flutter extensions

### Installation

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Get the dependencies:
   ```bash
   flutter pub get
   ```

3. Run the app:
   ```bash
   flutter run
   ```

## Features

- Simple counter functionality
- Navigation between pages
- Material Design 3 UI
- Clean and modern interface

## Project Structure

```
frontend/
├── lib/
│   └── main.dart          # Main application file
├── pubspec.yaml           # Dependencies and project configuration
├── analysis_options.yaml  # Linting rules
└── README.md             # This file
```

## Development

To add new features:
1. Create new widgets in the `lib/` directory
2. Update `pubspec.yaml` for new dependencies
3. Run `flutter pub get` to install new dependencies
4. Test your changes with `flutter run`
