# GetWork Web Frontend

- ใช้ TailwindCSS (`tailwind.config.js` อยู่ในโฟลเดอร์นี้)
- หน้าเว็บสามารถใช้ asset จาก `../mobile/assets`
- สามารถนำ component หรือ logic จาก `../mobile/lib/page`, `../mobile/widgets`, `../mobile/lib/main.dart` มาปรับใช้ในเว็บได้ (แต่ต้องแปลงเป็น HTML/JS/TS เอง)
- ไม่มีการเชื่อมต่อ API จริง
